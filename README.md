# To-do-list

Lista de tarefas básica

Funcionalidades:
- Adicionar novas tarefas;
- Remover tarefas existentes;
- Marcação de tarefas concluídas;
- Botão de remoção de todas as tarefas concluídas;
- Contador de tarefas concluídas/total de tarefas.

Características:
- Página principal em HTML;
- Estilos em CSS;
- Manipulação do DOM com JavaScript;
- Utilização de Local Storage do navegador para salvar alterações;
- Utilização de programação assíncrona ao inserir tarefas apenas como exemplo.
